# Chrome browser extension to collect search results from search engines
This browser extension can be used to collect search results from search engines like Google to use them in the Result Assessment Tool (RAT) or for other research.

How to install?
1. Download & Unzip:
Download the source code..

2. Open Extensions Menu:
Open Google Chrome and type chrome://extensions in the address bar (or go to Menu → Extensions → Manage Extensions).

3. Enable Developer Mode:
Toggle the switch Developer mode in the top right corner of the page.

4. Load Extension:
Click the button  Load unpacked that appeared in the top left.

5. Select Folder:
Select the folder you extracted in step 1. The extension should now appear in your list and is ready to use.
 
Note: Since this is a research tool, it is not listed in the official Chrome Web Store and must be installed manually using these steps.
